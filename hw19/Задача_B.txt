1. Что такое модификаторы доступа и зачем они нужны? -- Модификаторы доступа позволяют скрыть отдельные части кода

2. Какие бывают модификаторы доступа? - Private, Public, Defoult, Protected

3. Что такое package, для чего они нужны? - Чтобы группровать код. Это как папки для определённых вещей: фильмы, игры, музыка и т.д.

4. В чем отличие между default и public? - При public код будет доступен для всех классов внутри проекта. При default лишь внутри пакета.

5. Зачем делать сеттер private а геттер public, что это нам даст? - защита данных, не даст юзеру их перезаписывать. Юзер сможет лишь считать их
